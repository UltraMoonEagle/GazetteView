# GazetteView

Browse and search all statutory forms from the Occupational Safety, Health and Working Conditions (Central) Rules, 2026.

## Usage

1. Open the app and upload your Gazette PDFs (or paste extracted text)
2. Click **Extract all forms** — the index is saved locally and persists across refreshes
3. Click any form to view its rule reference and ask questions via AI
4. Set your [OpenRouter](https://openrouter.ai/keys) API key for Q&A (free, no credit card)

AI answers are cached in your browser — the same question is never sent to the API twice.

## Deploy

Push to GitHub and enable Pages (Settings → Pages → `main` branch, `/ root`).
